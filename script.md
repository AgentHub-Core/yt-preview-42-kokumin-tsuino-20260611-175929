# 42本目 約6分台本 v3

タイトル案: 【漫画】400万円差？追納しないと損…ねんきん定期便で青ざめた“10年切れ”の現実

作成日: 2026-06-11

目的: v2の会話量と制度説明は大きく削らず、1場面あたりの発話数を減らすためにシーンを細分化する。各画像は、その場面で話している人物の表情を主役にする。

## 制作ルール

- 0秒は主人公が年金事務所で驚愕する場面から入る。
- 会話を無理に短くしない。長い会話は、場面を分けて画像を増やす。
- 1シーンは原則2から5発話で切る。
- 1シーンの画像は、そのシーンで一番大事な発言をする人物の表情をメインにする。
- 画像内文字は1から3個まで。制度説明を画像内文字で詰め込まない。
- 免除、納付猶予、学生納付特例を混同しない。
- 追納が全員に必ず得とは言わない。期限、追納額、加算額、控除、年金見込み、今の家計で判断する。

## 登場人物

- さちこ: 61歳。美魔女系。上品で若々しいショートボブ、淡い色のカーディガン、薄い口紅。元スーパーの惣菜担当。家計が苦しい時期に国民年金の免除・納付猶予を受け、「承認されたなら老後も大きく減らない」と思っていた。
- たかし: 63歳。夫。イケオジ系。白髪交じりの短髪、整った眉、渋い目元。内装会社で働いてきた。12年前に仕事が減り、家計を立て直すために夫婦で必死だった。
- ゆい: 36歳。娘。可愛いけどしっかりしてる系。丸みのある目元、左目下の小さなほくろ、低い位置で結んだ髪。親の書類を責めずに並べ、確認順を決める。
- 美代子: さちこの元同僚。クセ強めの脇役。太い眉、赤い丸眼鏡、福耳、花柄スカーフ。口は軽いが、封筒と期限の確認だけは異様に早い。
- 年金事務所職員: クセ強めだが信頼できる脇役。小さな銀縁眼鏡、切れ長の目、きっちり分けた短髪。追納できる月、できない月、家計判断を分けて淡々と説明する。

美代子のスマホ通知と、赤い丸眼鏡を思い出して苦笑するさちこ。

[場面]
さちこのスマホに美代子からメッセージ。少しだけ空気がゆるむ。

[画像内文字]
- 封筒は寝かせるな
- 赤い眼鏡

[読む部分]
スマホ通知 美代子: 「だから言ったでしょ。封筒は寝かせるなって。」

さちこ: 「美代ちゃん、今それ言う？」

たかし: 「でも、あの人の赤い眼鏡、妙に忘れられないな。」

さちこ: 「悔しいけど、助かった。」

## S27 帰り道 10年は待ってくれない

[画像主役]
さちこ。視聴者へ向き直るような、真剣な表情。

[場面]
帰り道で、さちこが自分の失敗を言葉にする。

[画像内文字]
- 10年は待ってくれない
- 封筒を開けて

[読む部分]
さちこ: 「承認されたから安心、では終わらない。生活を守った後に、老後の金額も見に行かないといけなかった。」

ゆい: 「今から見る人は、まだ間に合う月があるかもしれない。」

さちこ: 「ねんきん定期便を見て、あれ、少ないと思ったら、怖くても封筒を開けてください。」

さちこ: 「10年は、待ってくれません。」

## S28 解説 免除や猶予は大事な制度

[画像主役]
落ち着いた表情のさちこ。背景に大きく「未納とは違う」。

[場面]
解説パート。人物の表情を残し、文字は詰め込まない。

[画像内文字]
- 未納とは違う
- 生活を守る制度

[読む部分]
ナレーション: 「今回のポイントです。」

ナレーション: 「一つ目。国民年金の免除、納付猶予、学生納付特例は、未納とは違います。生活が苦しい時に大事な制度です。」

## S29 解説 資格と金額は別

[画像主役]
職員。銀縁眼鏡で「資格」「金額」の2枚カードを示す。

[場面]
制度の違いを、細かい表ではなく2枚のカードで見せる。

[画像内文字]
- 資格
- 金額

[読む部分]
ナレーション: 「二つ目。ただし、受給資格に入ることと、将来の年金額に反映されることは同じではありません。」

ナレーション: 「免除期間は、満額納付と同じ年金額になるとは限りません。納付猶予や学生納付特例の期間は、追納しないと老齢基礎年金額に反映されません。」

## S30 解説 10年以内と家計判断

[画像主役]
ゆい。チェックリストを持ち、視聴者へ確認を促す表情。

[場面]
最後の確認。文字は3点だけにする。

[画像内文字]
- 記録を見る
- 10年以内
- 年金事務所へ

[読む部分]
ナレーション: 「三つ目。追納できるのは、追納が承認された月の前10年以内の免除等期間です。古い期間から順に、追納できる月とできない月を確認しましょう。」

ナレーション: 「四つ目。追納が必ず全員にとって得とは限りません。対象年度の追納額、加算額、社会保険料控除、年金見込み額、今の家計を合わせて判断してください。」

ナレーション: 「ねんきん定期便を見て、思ったより少ないと感じた人は、封筒を閉じないでください。ねんきんネットで記録を見て、分からないところは年金事務所で確認してください。」

ナレーション: 「10年切れは、気づいた時には戻せない月があります。今日見ることが、将来の大きな差を防ぐ第一歩です。」
