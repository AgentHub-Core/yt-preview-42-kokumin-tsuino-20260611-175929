# 42本目 約6分台本 v1

タイトル案: 【漫画】「免除なら減らない」と信じた61歳妻…10年の追納期限を過ぎて年金を増やせなかった理由

作成日: 2026-06-11

目的: 42番の新規テーマ「国民年金追納10年」を、制度解説ではなく、家族を守るために免除・猶予を選んだ主人公が、後年になって「受給資格」と「年金額」の違いに気づく物語として見せる。旧作業場の20枚画像素材とも対応させる。

## 制作ルール

- 台本の0秒は主人公の発言から始める。
- 冒頭30秒で「思ったより少ない」「追納は10年以内」を出す。
- 免除・納付猶予・学生納付特例を混同しない。
- 「免除なら満額」「全部無理」と断定しない。追納できる月とできない月を確認する流れにする。
- 追納が常に正解とは言わない。家計、控除、年金見込み、期限で判断する。
- 画像内文字は短く、読む部分では制度名と行動理由を省略しない。

## 登場人物

- さちこ: 61歳。主人公。美魔女系。上品で若々しいショートボブ、淡い色のカーディガン、穏やかな目元だが家計の数字を見ると一気に表情が鋭くなる。元スーパーの惣菜担当。家計が苦しい時期に国民年金の免除・納付猶予を受け、「承認されたなら大丈夫」と思っていた。几帳面だが、家族の前では弱音を飲み込む。
- たかし: 63歳。夫。イケオジ系。白髪交じりの短髪、整った眉、渋い目元、作業着でも清潔感がある。小さな内装会社で働いてきた。12年前に仕事が減り、家計を立て直すために夫婦で必死だった。
- ゆい: 36歳。娘。親の書類を一緒に整理する現実派。責めずに、確認順を決める。
- 美代子: さちこの元同僚。少し口は軽いが、書類だけは早めに確認するタイプ。
- 年金事務所職員: 淡々としているが丁寧。追納できる月、できない月、家計判断を分けて説明する。
